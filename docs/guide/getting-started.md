# 快速开始

## 环境要求

- Ubuntu 18.04 / 20.04 / 22.04
- 8 核处理器，16GB 内存
- NVIDIA Turing+ GPU 或 AMD GFX9/RDNA/CDNA
- Docker

## 获取源码

```bash
git clone https://github.com/ApolloAuto/apollo.git
cd apollo
```

## Docker 开发环境

```bash
bash docker/scripts/dev_start.sh
bash docker/scripts/dev_into.sh
```

## 构建

```bash
./apollo.sh build
```
